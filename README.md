# face-detector-api
backend for the face-detector app,


## link to the demo:
https://face-detector-cool-app.herokuapp.com/

## to get started 
1. clone the repo or download the zip file 
2. run "npm install"
3. run "npm start"

## tools used via npm
1. node.js 
2. express 
3. nodemon
4. knex 
5. bcrypt
6. clarifai API (FACE_DETECT_MODEL)
